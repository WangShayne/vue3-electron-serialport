## 环境需要：
node v14 64位

visual studio 2017 C++工作负载

python3.9

电脑设备上有COM2和COM3口
```
npm i -g node-gyp
```
## 查看环境是否满足
```
node-gyp list
```

## 安装依赖
```
npm i
```
出错的话就用`yarn`，一直重复安装，直到rebuild成功为止

## 启动
```
npm run serve
```
